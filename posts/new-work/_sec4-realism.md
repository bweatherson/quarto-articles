## Accuracy for Anti-Realists {#sec-realism}

Joyce's accuracy dominance argument is powerful, but it rests on a crucial assumption that anti-realists can reject. The argument works by showing that for any non-probabilistic credence function, there exists a probabilistic function that dominates it in accuracy—that is, scores strictly better by Brier score at every possible world. I'll argue that this dominance claim only goes through if we restrict ourselves to *complete* possible worlds. Once we allow incomplete possibilities, as the anti-realist about fiction should, the dominance argument fails. Non-probabilistic belief functions can be optimal credence functions.

Let me illustrate with a concrete case. Suppose I assign credences as follows: 1/3 to $p$, 1/3 to $\neg p$, and implicitly 1/3 to "neither"—a mass assignment characteristic of Dempster-Shafer functions. Joyce's argument suggests I should instead assign credences 1/2 to $p$ and 1/2 to $\neg p$, balanced probabilistically. At a complete possible world where $p$ is true, my Brier score is $(1/3 - 1)^2 + (1/3)^2 = 4/9 + 1/9 = 5/9$. The probabilistic rival scores $(1/2 - 1)^2 + (1/2)^2 = 1/4 + 1/4 = 1/2 = 4.5/9$. The probabilistic function does better. At a world where $p$ is false, I score $(1/3)^2 + (1/3 - 1)^2 = 1/9 + 4/9 = 5/9$, while the probabilist scores 1/2. Again, the probabilist wins. This is exactly Joyce's point: at every complete world, the probabilistic function dominates.

But now consider a possibility—an incomplete world in Humberstone's framework—where neither $p$ nor $\neg p$ holds. This possibility is incomplete because the question of whether $p$ is true is simply left unsettled. In the actual world of a fiction, or in any situation where realism about the domain fails, such incomplete possibilities are exactly what we should expect. At such a possibility, how should we score credences?

The most natural approach assigns a score based on how close the credence is to the truth value(s) that actually hold. When neither $p$ nor $\neg p$ is true at the possibility, I'll score each of my credences—1/3 in $p$ and 1/3 in $\neg p$—against the falsity of both. My Brier score is $(1/3)^2 + (1/3)^2 = 1/9 + 1/9 = 2/9$. The probabilistic rival, with credence 1/2 in each, scores $(1/2)^2 + (1/2)^2 = 1/4 + 1/4 = 1/2$. Since $2/9 \approx 0.222$ and $1/2 = 0.5$, my non-probabilistic credence function scores strictly *better* at this incomplete possibility.

The dominance is broken. At complete worlds where $p$ is true or false, the probabilistic function wins. But at the incomplete possibility, my belief function wins. There is no credence function that dominates the (1/3, 1/3) assignment everywhere—not even the probabilistic one.

This is the anti-realist's response to Joyce. The accuracy dominance argument assumes that accuracy matters *at all possible worlds*, and it implicitly assumes these worlds are *complete*. But an anti-realist about some domain—about fiction, or about properties that depend on disputed background assumptions—should not accept that there are complete possible worlds for every situation. Instead, there are incomplete possibilities where the truth is genuinely indeterminate.^[I should note that the anti-realist needn't be an incompletist about *reality itself*. The claim is specific: in the semantics of fiction, or in any domain where one is anti-realistic, completeness shouldn't be assumed. A realist about the external world and all concrete matters might still be an anti-realist about fiction, and so should work with possibility semantics for fiction.]

More generally, for any belief function (any Dempster-Shafer assignment of mass to sets of possibilities), there exists a frame of possibilities—a set of incomplete possibilities—relative to which that belief function is optimal in accuracy. Belief functions are not second-class credences that fail to maximize accuracy across all complete worlds; rather, they are exactly the credence functions that maximize accuracy relative to appropriate *incomplete* possibility frames. If your domain is one where realism fails, incomplete possibilities are appropriate, and so non-probabilistic credences are appropriate.

This connects to earlier work by Horowitz on guessing and accuracy. The guessing framework makes explicit what seems to be assumed in standard accuracy dominance arguments: that at every possible world, every proposition is either true or false, and you're being evaluated on how well you guess the complete truth. But if you're not trying to guess a complete truth—if the domain is one where some matters are genuinely indeterminate—then guessing games and the dominance arguments built on them no longer apply in the same way.^[See [@Horowitz] on the guessing framework. The framework is elegant but presupposes realism: that there is a complete fact of the matter to be right about.]

Several objections deserve brief attention. First, someone might say: "This doesn't actually save Dempster and Shafer themselves. Their work wasn't framed in anti-realist terms." That's correct, and I don't claim otherwise. Dempster-Shafer functions are useful for anti-realists, but anti-realism is not what originally motivated their development. The point is that anti-realists have good reason to work with belief functions, not that this was always the intended application.

Second, the approach seems to rest on an unusual account of disjunction. How can it be that $p$ is false, $\neg p$ is false, and yet these are the relevant propositions at a possibility? This does seem strange from a classical perspective. But I'd argue that disjunction *is* strange, and we've underinvested in understanding it.^[It's curious that the philosophical literature contains roughly one hundred times as many papers and books on conditionals as on disjunction. If disjunction is this philosophically important—as it must be if it's central to how possibilities are structured—we should think more carefully about it. Humberstone's encyclopedic *The Connectives* devotes a substantial chapter to possibilities as devices for understanding disjunction, which suggests this is neither new nor fringe.] A well-developed account of disjunction that respects the structural role of possibilities would vindicate the anti-realist approach.

Third, one might protest that this is all rather specific to fiction, or to trivial questions like what's true in a story. But the scope of the argument is wider than that. Any *substantive* question about a fiction—whether Jane Austen's *Emma* really endorses Emma's matchmaking, for instance—should admit of incomplete possibilities where there simply is no fact of the matter. Hard questions about fiction require taking seriously the possibility that the text underdetermines the truth. That possibility, lodged in the possibility frame, naturally induces non-probabilistic credences.

Moreover, the problem is not restricted to fiction. Any domain where realism might fail is a candidate for anti-realist treatment. This could include ordinary questions where our evidence is thin. Consider skeptical scenarios: we might suspend between realism about the external world and skeptical scenarios where it doesn't matter whether the world is as it appears. In low-evidence situations, the evidence might be silent on whether realism obtains. This is where Pryor's response to White on dogmatism becomes relevant.^[White [@White2006] argues that you cannot gain justification from a perceptual appearance without having prior justification. Pryor [@Pryor2007] argues that if the evidence from appearance is itself silent on realism vs. skepticism, you can maintain equal credence in both without any prior justification—a kind of equanimity that blocks White's regress. The anti-realist framework here provides additional structure to that idea.] If we cannot rule out realism-undermining scenarios, we should not assign credence functions that presuppose realism, but should use belief functions that remain neutral across the possibility of fundamental indeterminacy.

Finally, someone will ask: "If all this is true, why is probabilism so unreasonably effective?" The answer is that probabilism is unreasonably effective *given realism*. In domains where we can safely assume that a complete truth exists, probabilism is the right credence framework. This is structurally similar to a point Priest has made about logic: classical logic is the correct logic for reasoning about situations that are known to be consistent. I say much the same: probabilism is the correct credence framework for situations that are known to be complete. But when completeness is not guaranteed—when realism is in question—belief functions become not just permissible but rationally optimal. The accuracy dominance argument, properly understood, gives us no reason to reject them.
